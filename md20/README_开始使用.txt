  财务管理系统 - Windows 快速开始指南
═══════════════════════════════════════════════════════════

🚀 最简单的开始方式：

1. 双击运行 "快速开始.bat"
2. 按照菜单提示操作：
   - 选择 1：检测环境
   - 选择 2：安装 XAMPP（如果没有安装）
   - 选择 3：配置数据库
   - 选择 4：启动服务器
   - 选择 5：访问系统

═══════════════════════════════════════════════════════════

📋 详细步骤：

步骤1: 安装 XAMPP（如果没有）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方式A: 自动下载（推荐）
  - 运行 "快速开始.bat"，选择选项 2
  - 或运行 "download_xampp.ps1"（需要 PowerShell）

方式B: 手动下载
  - 访问: https://www.apachefriends.org/download.html
  - 下载 XAMPP for Windows (PHP 8.1)
  - 运行安装程序，安装到 C:\xampp

步骤2: 配置数据库
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - 运行 "快速开始.bat"，选择选项 3
  - 或直接运行 "setup.bat"
  - 输入 MySQL root 密码（XAMPP 默认密码为空，直接回车）

步骤3: 启动服务器
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - 运行 "快速开始.bat"，选择选项 4
  - 或直接运行 "start_server.bat"
  - 或手动打开 XAMPP Control Panel，启动 Apache 和 MySQL

步骤4: 访问系统
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - 运行 "快速开始.bat"，选择选项 5
  - 或浏览器访问: http://localhost/public/index.php?r=auth/login
  - 默认账号: admin / admin123

❓ 常见问题：

Q: 提示"未找到 XAMPP"
A: 请先安装 XAMPP，运行"快速开始.bat"选择选项 2

Q: MySQL 无法启动
A: 检查端口 3306 是否被占用，或查看 XAMPP 错误日志

Q: Apache 无法启动
A: 检查端口 80 是否被占用，可在 XAMPP 中修改端口

Q: 数据库连接失败
A: 确保 MySQL 服务正在运行，检查 app/config/db.php 配置

Q: 页面显示 404
A: 确认访问路径正确: public/index.php?r=auth/login
